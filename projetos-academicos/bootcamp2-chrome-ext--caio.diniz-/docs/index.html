Tab Closer — Extensão Chrome MV3

Sobre o Projeto

Extensão Google Chrome que fecha todas as abas abertas, exceto a ativa, para limpar rapidamente sua janela.

Instalação Manual

  1. Baixe o projeto do GitHub.
  2. Acesse chrome://extensions no Chrome.
  3. Ative o modo desenvolvedor (Developer Mode).
  4. Clique em "Carregar sem compactação" e selecione a pasta do projeto.

Funcionalidades

  • Interface simples e moderna.
  • Fecha todas as abas, menos a atual.
  • Comunicação eficiente entre popup e background.
